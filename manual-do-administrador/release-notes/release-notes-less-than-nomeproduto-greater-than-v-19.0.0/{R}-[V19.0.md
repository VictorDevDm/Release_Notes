# Desabilitar Configurações na Tela de Bloqueio no Work Profile

**ID da US: US-NPD-7730**

Descrição: No portal, ao acessar a seção de políticas "Android - Work Profile", você tem a opção de desativar as configurações da "Tela de Bloqueio" na aba “Configurações”. Isso dá a você mais controle sobre como o Perfil de Trabalho é gerenciado no dispositivo. Quando você salva a política após fazer essa alteração, as novas configurações são automaticamente enviadas para todos os dispositivos que estão conectados a essa política, garantindo que todas as mudanças sejam aplicadas de forma eficiente e rápida.
